# Project index

Read this file first when tailoring cover letters or answering application
questions. Each entry says what the project is and when to cite it, so you can
open only the 1-2 READMEs that fit the job instead of loading the whole folder.

`about.md` is the profile summary (who Marcus is, focus areas), not a project.

## AI, automation, and data pipelines

- **policy-intelligence-engine.md** — AI Policy Assistant: enterprise RAG
  pipeline (Google Drive ingestion, Gemini embeddings, Pinecone, n8n
  orchestration, agentic retrieval with anti-hallucination guardrails).
  *Best single automation/AI example; lead with this for AI, RAG, LLM, and
  automation roles.*
- **auto-apply-bot.md** — This system: AI job-search agent (TypeScript, Claude
  API, n8n, SQLite, Playwright browser automation) with multi-stage LLM
  scoring, human approval gates, and safety rails. *Use for agentic AI,
  LLM-pipeline, browser-automation, and TypeScript backend roles.*
- **content-pipeline.md** — Strauss AI content pipeline: generator/evaluator
  loop (Python) with a fresh-context LLM judge, mechanical rubric checks,
  approval-gated publishing via n8n to LinkedIn/blog. *Use for LLM evals,
  content automation, and human-in-the-loop design.*
- **social-intelligence-engine.md** — Social Media Market Auditor:
  proxy-based scraping plus Gemini sentiment analysis and structured
  competitive-intelligence reports, event-driven. *Use for data pipelines,
  scraping, and analytics roles.*
- **ads-analytics-agent.md** — Read-only Claude agent workspace over Google
  Ads and GA4 (MCP servers, GAQL, keyword/ad-copy tooling) for Strauss AI and
  agency clients. *Use for marketing-analytics, MCP, and agent-tooling roles.*
- **sundial.md** — Shelved 2026-07-04: geospatial lead-gen toolkit (StatCan
  addresses, satellite tiles classified by Gemini, Google Solar API, eval
  harness with human grading). *The honest failure story: killed after
  calibration showed a weak product. Use for "tell me about a failure" or
  eval-rigor questions; do not pitch as an active product.*

## Developer tools and games

- **anti-slop.md** — Mac Electron app for running multiple AI coding agents
  (Claude Code, Codex, Gemini CLI) side by side with context staging and
  routing. *Use for AI dev-tooling, Electron, and multi-agent UX roles.*
- **gitcraft.md** — Git-like version control for Minecraft builds: Java 21
  Paper plugin, JGit push/pull to GitHub, SQLite, branching/merge/diff with
  in-game conflict previews. *Use for Java, plugin/API, and systems-design
  roles; also a strong "novel idea" story.*
- **iso-city.md** — Isometric city-builder/logistics/warfare game in
  TypeScript: truck and rail logistics networks, supply-driven combat, enemy
  AI playing by player rules. *Use for game dev, simulation, and complex
  front-end state roles.*
- **dotfiles.md** — Public macOS/Linux terminal setup (zsh, Neovim with local
  AI completion via Ollama, Ghostty) with a cross-platform installer. *Minor;
  use as evidence of tooling craft and open-source habits.*
